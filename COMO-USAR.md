# 🚀 Cómo usar las skills

Cada skill se ejecuta como un **comando de barra (slash command)**. En el chat de
Claude escribe **`/`** para ver la lista de skills instaladas y selecciona la que
necesites, o escribe el nombre directamente (por ejemplo `/constructor-de-voz`).
Después de llamarla, dale el contexto que te pida.

> 💡 Truco: escribe solo **`/`** en el chat y aparecerá el menú con todas las
> skills disponibles. Así no necesitas memorizar los nombres.

---

## Paso 1: Construye tu voz (hazlo una sola vez)

Empieza ejecutando:

```
/constructor-de-voz
```

Claude te hará algunas preguntas y te pedirá **3 a 5 ejemplos** de cosas que ya
hayas escrito (publicaciones, correos, artículos… lo que sea). Con eso aprende
tu tono y tu estilo, y los guarda para usarlos siempre.

👉 Ten a mano esos ejemplos antes de empezar. Si no tienes ninguno, escribe
*"usa ejemplos"* y Claude cargará unos de muestra que podrás reemplazar después.

> **¿Por qué primero?** Todas las demás skills usan tu voz. Si te saltas este
> paso, el contenido sonará genérico.

---

## Paso 2: Llama la skill que necesites

Ejecuta el comando y luego describe tu pedido o pega tu material. Estas son las
skills agrupadas por objetivo:

### ✍️ Escribir publicaciones
| Comando | Para qué |
|---|---|
| `/redactor-de-publicaciones` | Escribe una publicación de LinkedIn en tu voz |
| `/formateador-de-publicaciones` | Convierte un tema en publicación con una estructura probada (PAS, AIDA…) |
| `/generador-de-ganchos` | Genera 6 ganchos para empezar fuerte |

## Un flujo de trabajo completo (ejemplo)

Así se ve usar el sistema de principio a fin en una semana:

1. `/investigacion-de-nicho` → obtienes una lista de temas relevantes.
2. `/redactor-de-publicaciones` → eliges un tema y Claude escribe el borrador en tu voz.
3. `/evaluador-de-publicaciones` → Claude te dice qué tan fuerte es y qué mejorar.
4. `/carrusel-gemini` → creas un visual para acompañar el texto.
5. `/comentario-fijado` → preparas un comentario para sumar alcance al publicar.

Repites el ciclo y, cada cierto tiempo, usas `/panel-de-analiticas` para ver qué
está funcionando y ajustar.

---

## Consejos para mejores resultados

- **Ejecuta `/constructor-de-voz` primero.** Sin tu voz, el resto suena genérico.
- **Da contexto después de llamar la skill.** Pega tus notas, un enlace o el tema
  de tu newsletter; mientras más material, mejor el resultado.
- **Itera sin miedo.** Si algo no te convence, dile *"hazlo más corto"*, *"más
  directo"* o *"con otro gancho"*. Claude ajusta.
- **Revisa antes de publicar.** Las skills te dan un excelente punto de partida;
  el toque final siempre es tuyo.
